# Contributor Covenant Code of Conduct

Participation must be harassment-free. Welcome differing viewpoints, accept constructive feedback, and respect privacy. Harassment, insulting comments, and publishing private information without permission are unacceptable. Maintainers may remove contributions that violate these standards. This document is adapted from Contributor Covenant 2.1: https://www.contributor-covenant.org/version/2/1/code_of_conduct.html.
